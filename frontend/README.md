# 🥒 Pickle Frontend

Pickle Frontend는 AIoT 스마트 쇼핑카트 서비스의 **사용자 인터페이스 전반을 담당하는 레이어**로,  
웹 키오스크(Vue)와 모바일 앱(Flutter)을 통해 카트 디바이스 및 백엔드와 연결됩니다.

Frontend는 단순 화면 출력이 아니라,  
**카트 세션 연결 → 상품 인식 결과 시각화 → 결제 흐름 제어 → 소비 데이터 시각화**까지  
서비스 전반의 UX 흐름을 책임집니다.

---

### 🎯 담당 영역

#### 🖥 Web Kiosk (Vue)

- 카트 디바이스와 연결된 세션 상태 실시간 표시
- Edge AI 상품 인식 결과 시각화 (상품명 / 수량 / 금액)
- 장바구니 합산 금액 실시간 갱신
- 결제 진행 상태 모니터링
- 관리자용 상품 / 카테고리 관리 UI (선택 기능)

#### 📱 Mobile App (Flutter)

- 사용자 인증 (로그인 / 회원가입)
- QR 기반 카트 세션 연결
- 결제 수단 등록 및 관리
- 장바구니 및 결제 내역 조회
- 가계부 및 소비 분석 UI 제공
- 구매 상품 기반 레시피 추천 UX

---

### 🔄 Frontend 중심 서비스 흐름

1. 사용자가 모바일 앱에서 로그인
2. QR 스캔으로 카트 세션 연결
3. 상품 담기 → Edge AI 인식
4. Web Kiosk에서 상품 리스트 및 합산 금액 실시간 반영
5. 카트 디스플레이와 상태 동기화
6. 모바일 앱에서 결제 승인 확인
7. 결제 완료 후 가계부 / 소비 분석 화면 제공
